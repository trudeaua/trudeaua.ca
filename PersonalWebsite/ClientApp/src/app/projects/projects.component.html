Cosmic Quest: Order & Chaos

Description

Tour Guide

Description
  • Tour Guide allows you to generate a shortest-path tour route around a start location, and select a number of different categories of locations that you would like to see around that city.
  • The tour route generation algorithm uses Dijkstra's shortest path algorithm to compute the shortest path between all points on the route.
  • This project served as an introduction to UI design, Java application development, and algorithms.
Developed With: Java, JxMaps, Java Swing, Google APIs
